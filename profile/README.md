### How we build

| Principle | What it means |
|:--|:--|
| **Lean & Fast** | Every model optimized for speed and cost. No bloated infrastructure, no wasteful compute. |
| **Your Data, Your Rules** | DPDPA-compliant by design. We never train on customer data. Period. |
| **Safe by Default** | Built-in guardrails, bias testing, and content filtering on every product we ship. |
| **Open by Choice** | Built on MCP and open-source frameworks. No vendor lock-in, full interoperability. |
